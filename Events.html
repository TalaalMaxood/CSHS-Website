Monthly Programming Problems

Participation

Code is due on Friday, November 11th at 4:00PM

For this months's programing problem, we want you to create code that can make Pascal's triangle

The code should take an input n and be able to ouput n rows of Pascal's Triangle

    e.g with an input of 5, the program should show the following

    1

    1 , 1

    1 , 2 , 1

    1 , 3 , 3 , 1

    1 , 4 , 6 , 4 , 1

You can earn 1 participation point for a working program. If you were unable to finish the program but showed effort, you can earn 0.5 points. Please do not cheat and violate the CSHS Honor Code over 1 participation point.
